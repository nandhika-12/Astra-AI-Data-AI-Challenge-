# Astra AI 🚀

## AI-Powered Personal Growth & Career Assistant

Astra AI is an intelligent AI assistant designed to help students and young professionals improve their academic performance, career readiness, productivity, and personal growth through personalized guidance and AI-driven insights.

---

## 📌 Problem Statement

Students often struggle with:

* Lack of personalized career guidance
* Difficulty identifying skill gaps
* Limited access to mentors
* Poor interview preparation
* Unstructured learning paths
* Scattered resources across multiple platforms

As a result, students spend more time searching for information than actually learning and growing.

---

## 💡 Our Solution

Astra AI serves as a one-stop intelligent assistant that combines career mentoring, resume analysis, interview preparation, learning roadmaps, goal tracking, and productivity support into a single platform.

The system leverages Generative AI and personalized recommendations to guide users through their academic and professional journey.

## 🏗 System Architecture

```text
User
  ↓
Frontend (React / Web App)
  ↓
Astra AI Interface
  ↓
Backend API (Python / FastAPI)
  ↓
OpenAI / Gemini API
  ↓
Supabase Database
  ↓
Personalized Response
```

## 🔄 Workflow

1. User creates an account and logs in.
2. User interacts with Astra AI through chat.
3. User uploads a resume or document.
4. AI analyzes user data and generates insights.
5. Personalized recommendations are provided.
6. Goals and progress are tracked continuously.
7. Users improve skills through guided learning and mock interviews.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/astra-ai.git
cd astra-ai
```

### Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### Install Frontend Dependencies

```bash
npm install
```

### Start Backend

```bash
python app.py
```

### Start Frontend

```bash
npm start
```
